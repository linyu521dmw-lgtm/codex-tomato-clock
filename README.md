# Pomodoro Widget

这是一个适合桌面端和手机端使用的番茄钟专注小组件。

## 项目简介

Pomodoro Widget 是一个简洁的专注计时工具，支持专注、短休息和长休息三种模式。界面以番茄视觉为核心，适合产品设计学生用于练习网页小组件、交互逻辑和移动端界面展示。

## 核心功能

- 专注模式：25 分钟
- 短休息：5 分钟
- 长休息：15 分钟
- 开始、暂停、重置
- 今日完成次数统计
- 手机端和电脑端自适应显示
